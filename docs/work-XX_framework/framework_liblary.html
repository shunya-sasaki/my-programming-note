フレームワークとライブラリの違い

フレームワーク

  • アプリケーションを呼び出す

  • 型にはめるたの枠組み

  • 全体の処理の流れは実装済

  • 開発者が処理の一部を埋める

  • 例) Tensorflow

  • ディレクトリ構造が決まっている

  • 特別なファイルがある->_app.jsとか_document.jsとか

ライブラリ

  • アプリケーションに呼び出される

  • 便利な機能を持つ部品群

  • 全体の処理の流れやロジックはない

  • 必要な時に必要なものだけ使うことができる

  • UIを作るためのライブラリ=部品群

  • 1コンポーネント1ファイルから使える
